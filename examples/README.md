# Examples

This directory contains example outputs from analyzing the **SWE-Compass** paper (arXiv 2511.05459v3) with Paper Lens.

## Contents

```
swe-compass/
├── speed-read.md       # Speed Read mode output
├── deep-learn.md       # Deep Learn mode output
└── slides-content.md   # Present mode output
```

## What each file shows

- **speed-read.md** — A ~5 minute digest: paper metadata, TL;DR, core contributions, method overview, key experimental results, and Q&A.
- **deep-learn.md** — An interactive deep-dive: includes a term glossary with plain-language explanations, step-by-step method breakdown, formula walkthroughs, appendix highlights, and practical takeaways.
- **slides-content.md** — A 20-slide presentation outline with speaker notes, figure references, and time allocation.

## Not included

- **images/** and **figures/** directories are omitted (too large for the repository).
- **paper.pdf** is not included due to copyright. You can download the paper from [arXiv](https://arxiv.org/abs/2511.05459v3).
- **presentation.html** is not included (depends on the frontend-slides companion skill and the extracted figures).

## Try it yourself

To reproduce these outputs, open Claude Code and run:

```
> Help me read this paper: https://arxiv.org/pdf/2511.05459v3
```
